# Harmonica Video Editor

React/Vite renderer with an Electron desktop shell.

## Development

- `npm run dev` runs the web app in Vite.
- `npm run dev:electron` runs the Vite dev server and launches Electron against it.

## Production build

- `npm run build` builds the web renderer into `dist/`.
- `npm run build:electron` builds the renderer and packages the Electron app with `electron-builder`.
